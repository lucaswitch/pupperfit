# Pupperfit

## Introduciton

Pupperfit is a webservice that responds incoming url requests for printing PDF pages as 

## Installation
  With npm
  ```
    npm install pupperfit     
  ```
  With yarn
  ```
    yarn add pupperfit
  ```
## Running the service

  With npm
  ```   
    npm run    
  ```
  With yarn
  ```
    yarn run
  ```
## Printing a HTTP page

 Create a post request to http://localhost:3000 passing a url as data.
